Getting started

How to use this tool

Instructions in three languages. Choose yours below.

Step 1

Enter your API key when prompted. You only need to do this once — it saves in your browser.

Get a key at console.anthropic.com
Step 2

Choose a tab: Text for typing or pasting, Audio for recordings, Import for documents.

Step 3

Select your languages, domain (General, Business, Academic, Legal, or Thai Official Letter), and any optional features.

Step 4

Click Run and wait a few seconds. Copy the result or use the Send to Text tab button to refine further.

ขั้นตอนที่ 1

กรอก API key เมื่อระบบแจ้ง ทำเพียงครั้งเดียว — ระบบจะจำไว้ในเบราว์เซอร์ของคุณ

Administrator setup

One-time setup for your team

Follow these steps once to make the tool available to all InEd staff.

1
Anthropic

Get an API key

Go to console.anthropic.com, create an account, and generate an API key. This is what powers the AI translations.

sk-ant-api03-…

Estimated cost: $2–5 USD/month for a small team

2
GitHub

Create a GitHub repository

Sign up at github.com, create a new repository called ined-tool, and upload both downloaded HTML files.

Rename ined-trilingual-tool.html to tool.html

3
GitHub Pages

Enable free hosting

In your repository, go to Settings → Pages, set Source to Deploy from a branch, select main branch, and click Save.

Your tool will be live at:

https://[username].github.io/ined-tool/
4
Wix

Embed in your Wix site

In Wix Editor: Add (+) → Embed & Social → HTML iframe. Paste your GitHub Pages URL. Set iframe height to 900px.

Tip: add the page to a Members-only section so only staff can see it

5
Staff

Share with your team

Share the Wix page URL and your API key with staff. They paste the key once on first visit — their browser remembers it.

Use a secure channel (Signal, encrypted email) to share the key

6
Optional

Custom domain

If InEd has a custom domain, you can point it to GitHub Pages for a cleaner URL like tools.inedfoundation.org.

Settings → Pages → Custom domain in GitHub

Questions

Frequently asked

Common questions from InEd staff.

Is my text kept private? ▼
Your text is sent to Anthropic's servers for processing and is subject to Anthropic's privacy policy. Anthropic does not use API data to train models. Do not enter passwords, ID numbers, or highly sensitive personal data.
How accurate is Burmese translation? ▼
Claude (the AI model) uses Unicode Burmese and performs well on standard written Burmese. For sensitive documents or legal/official letters, always have a native speaker review the output before sending.
What is the "Thai Official Letter" domain? ▼
This mode applies the formal หนังสือราชการ (official government letter) register and format used by Thai government agencies. Useful when communicating with PESAO, MECC, or other official bodies.
Why do I need an API key? ▼
This tool is self-hosted, meaning InEd controls it entirely. The API key connects the tool to Anthropic's AI service. You pay only for what you use — typically $2–5 USD per month for a small team's usage.
The Google Drive fetch isn't working. ▼
The Google Drive import works when the file is accessible to your Google account. As a standalone hosted file, direct Google Drive API access requires additional OAuth setup. The easiest workaround: open the Google Doc, copy all text (Ctrl+A, Ctrl+C), and paste into the Text tab.
